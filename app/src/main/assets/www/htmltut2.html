HTML5 boilerplate

This is the basic html5 boilerplate

The markup contains the head tag which is responsible for the meta data of the page like title and other linked files like css and js files

The next tag body tag contains the overall content of the webpage that has to be displayed over the web

The link tag is used to link to resources such as css files to style the content
